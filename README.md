# Simulacion numérica Los Picos de Europa
Recompilación del código de python o matlab para resolver los ejercicios propuestos en clase durante el curso.

1.- **PVI_Euler** (21/01/26)

  resolución del problema de valor inicial mediante el método de Euler variando el parámetro h

2.- **aprox_ED** (26/01/26)

  aproximación de un problema de contorno variando el número de puntos interiores usados

3.- **aprox_func_red_neuronal** (28/01/26)

  aproximación de una función en 200 épocas comparando 2, 3 y 4 neuronas usando tangentes hiperbólicas

4.- **mod_pesos_extr_interv** (02/02/26)

  comparación de función de activación tangente hiperbólica / sigmoide en los extremos del intervalo

5.- **aprox_seno** (04/02/26)

  distintas aproximaciones de la función seno en distintos intervalos cerrados con distinto número de neuronas
